# Adidas Sales Dashboard (Power BI)

This project showcases an interactive sales dashboard built using Microsoft Power BI to analyze Adidas sales data. The dashboard provides key insights into revenue, profit, product performance, and regional sales trends.

## 🚀 Features
- 📊 Sales overview (Total Revenue, Profit, Units Sold)
- 🌍 Region-wise and country-wise performance
- 🛍️ Product category and retailer analysis
- 📅 Time-based trends (Monthly / Yearly)
- 🎯 KPI cards for quick insights
- 🔍 Interactive filters and drill-through analysis

## 🛠️ Tools & Technologies
- Microsoft Power BI
- Power Query (ETL)
- DAX (Data Analysis Expressions)

## 📈 Key Insights
- Identifies top-performing regions and products
- Tracks seasonal sales trends
- Helps in data-driven decision making

## 📂 Project Structure
- `Adidas_Sales_Dashboard.pbix` → Power BI report file
- `dataset/` → Source data files (if included)
- `images/` → Dashboard screenshots

## 🎯 Use Case
This dashboard can be used by business stakeholders to monitor sales performance, optimize strategies, and improve profitability
